# DingDong - The Future of Clean

Welcome to **DingDong**, a stunning, high-performance, and futuristic web application built with a premium "Deep Space" aesthetic. This project was built using **React**, **TypeScript**, **Tailwind CSS**, and **Framer Motion**, powered by the lightning-fast **Vite** bundler.

## Getting Started

Follow these simple steps to run this project on any computer:

### Prerequisites

You will need **Node.js** installed on your machine.
- Download and install it from [nodejs.org](https://nodejs.org/) (we recommend the LTS version).

### Installation & Running

1. **Clone the repository** (if you haven't already):
   ```bash
   git clone https://github.com/Shaazomar/Ding-Dong.git
   cd Ding-Dong
   ```

2. **Install the dependencies**:
   Run the following command in the root folder of the project to download all necessary packages:
   ```bash
   npm install
   ```

3. **Start the development server**:
   Once installation is complete, start the app:
   ```bash
   npm run dev
   ```

4. **View the Application**:
   Open your browser and navigate to the URL provided in the terminal, usually:
   ```
   http://localhost:5173/
   ```

## Production Build

To create an optimized production build, run:
```bash
npm run build
```
This will generate a `dist` folder containing the compiled assets ready for deployment.

## Tech Stack
- **Framework**: React 18
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Routing**: React Router DOM
- **Icons**: Lucide React
- **Build Tool**: Vite
